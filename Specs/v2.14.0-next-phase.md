  suite had 16 failures. Root cause: `autoresearch.sh` counted `N passed` while
  discarding pytest's exit code, and 16 tests failed on a single missing
  `import json` in `utag_generators/targets/multi_language_sdk.py` (fixed).
- README claimed "64 tests"; real count is 391 collected (374 pass, 17 env-skips:
  control-plane binary not built, postgres unavailable, no ANTHROPIC_API_KEY).

- `utag intel` had a dead duplicate `cmd_intel` definition (removed).
- `utag openapi` subcommands (`normalize`, `bundle`, `diff`, `overlay`, `lint`,
  `agent-readiness`) are **stub prints** — flagged as fake-feature debt, to be
