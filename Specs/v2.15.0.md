# Spec: v2.15.0 — Self-Generated Interactive Console, Real OpenAPI Pipeline, Live AI Adapters

Closes the four leftover clusters from `TODO/v2.14.0-next-phase.md`.

## Governing constraint

The console is 100% dogfood: every component, stylesheet, variable, fixture,
route, and interaction is generated from `design.yaml` by UTAG's own pipeline.
The only hand-written frontend file is `packages/ui/src/main.tsx` (17 lines)
plus build config. Anything a screen needs is a **generator upgrade**, never a
hand-written component.

## Decisions of record

1. **Component-type vocabulary supersedes the 11 planned UDC generators.**
   `react-component-library` renders types `card | table | matrix | panel |
   timeline | chart | board | graph | nav`, each a real deterministic
   accessible renderer over the contract bound via `design.yaml
   data.contracts`. Run-timeline, coverage-matrix, workflow-graph, task-board,
   cost-latency-chart etc. are design.yaml components, not separate generators.
2. **`data.contracts` (resource → schema kind)** drives typed data props,
   `generated/schemas/contracts.ts` emission, and schema-valid fixture
   generation. Screens with no matching contract fail at generation time — a
   design error, not a runtime surprise.
3. **Interactions are generated**: `design.yaml interactions:` becomes
   `generated/hooks/useInteractions.tsx`; sources call `select(id, record)`,
   targets read `useSelected(id)`. Route pages place interaction targets in
   the layout's inspector region.
4. **`utag openapi` stub prints retired** — real normalize/bundle/diff/
   overlay/lint/agent-readiness in `utag_core.openapi`, reporting through the
   strict `OpenApiOperation`/`OpenApiDiffReport`/`OpenApiReadinessReport` and
   `ValidationFinding` contracts. The stub-asserting tests were replaced.
5. **Live AI calls only through the router + adapter**: telemetry
   (`prompt-run`, `model-call-trace`, `utag.ai.call` span), policy budgets
   (breach = error finding + event + failed call; unknown cost under a cost cap
   = explicit warning), sha-keyed cache, and credential refusal for non-local
   providers. Ports without usage data record nulls, never invented numbers.
